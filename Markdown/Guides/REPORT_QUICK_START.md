## üéØ Quick Tips

### **For Best Results:**

1. ‚úÖ **Always use A4 paper size**
2. ‚úÖ **Keep scale at 100%**
3. ‚úÖ **Enable background graphics** (to show borders)
4. ‚úÖ **Disable headers/footers** (cleaner look)
5. ‚úÖ **Use latest browser version**

### **Common Issues:**

**Q: Logo doesn't show?**
- A: Enable "Background graphics" in print settings

**Q: Pages cut off?**
- A: Make sure paper size is set to A4

**Q: Text too small?**
- A: Don't change scale - keep at 100%

**Q: Can't see documents?**
- A: Check if files were uploaded during registration
